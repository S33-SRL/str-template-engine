String Teamplate Engine

This library allows you to "serialize" an untyped object into a string using a Template String as a model

Work with base tamplate

Render single field: "{{"{~property}"}}" Example: "{{"{code}"}}" | Result: {{tmpCalc("{code}")}}

Navigate fields : "{{"{~property.~sub-property}" }}" Example: "{{"Type: {type.description}" }}" | Result: {{tmpCalc("Type: {type.description}")}}

Render multi fields : "{{"{~property1} - {~property2}" }}" Example: "{{"{code} - {shortDescription}" }}" | Result: {{tmpCalc("{code} - {shortDescription}")}}

Render nasted fields : "{{"{{property1}~part-of-property2}" }}" Example: "{{"Main: {mainDescription} - Nasted: {{mainDescription}Description}" }}" | Result: {{tmpCalc("Main Descripion:{mainDescription} - Description Nasted: {{mainDescription}Description}")}}

Work with array

Render element of an array by index (starting from index 0): "{{"{~array[~index].~property}"}}" Example: "{{"{tags[1].code}"}}" | Result: {{tmpCalc("{tags[1].code}")}}

Render first element of an array: "{{"{~array[first].~property}"}}" Example: "{{"{tags[first].code}"}}" | Result : {{tmpCalc("{tags[first].code}")}}

Render last element of an array: "{{"{~array[last].~property}"}}" Example: "{{"{tags[last].code}"}}" | Result : {{tmpCalc("{tags[last].code}")}}

e Render first element in array (like js find): "{{"{~array[~property-key-value,~property1].~sub-property}"}}" Example: {{"Sell price: {prices[customer,kind].price} - Buy price: {prices[supplier,kind].price}"}} | Result: {{tmpCalc("Sell price: {prices[customer,kind].price} - Buy price: {prices[supplier,kind].price}")}}

Data Conversion

Date: "{{"{@Date|~date|~string-format}"}}" Example: "{{"{@Date|2016-06-10T15:19:25.75|DD/MM/YYYY}"}}" | Result: {{tmpCalc("{@Date|2016-06-10T15:19:25.75|DD/MM/YYYY}")}}
Example: "{{"{@Date|{creationDate}|YYYY-MM-DD HH:mm}"}}" | Result: {{tmpCalc("{@Date|{creationDate}|YYYY-MM-DD HH:mm}")}}

Bool: "{{"{@Bool|~variable}"}}" Example: "{{"{@Bool|}"}}" | Result: {{tmpCalc("{@Bool|}")}}
Example: "{{"{@Bool|duck}"}}" | Result: {{tmpCalc("{@Bool|duck}")}}
Example: "{{"{@Bool|false}"}}" | Result: {{tmpCalc("{@Bool|false}")}}
Example: "{{"{@Bool|true}"}}" | Result: {{tmpCalc("{@Bool|true}")}}
Example: "{{"{@Bool|undefined}"}}" | Result: {{tmpCalc("{@Bool|undefined}")}}
Example: "{{"{@Bool|{installationComponent}}"}}" | Result: {{tmpCalc("{@Bool|{installationComponent}}")}}

Number: "{{"{@Number|~variable}"}}" Example: "{{"{@Number|}"}}" | Result: {{tmpCalc("{@Number|}")}}
Example: "{{"{@Number|true}"}}" | Result: {{tmpCalc("{@Number|true}")}}
Example: "{{"{@Number|012}"}}" | Result: {{tmpCalc("{@Number|012}")}}
Example: "{{"{@Number|23.342}"}}" | Result: {{tmpCalc("{@Number|23.342}")}}
Example: "{{"{@Number|{leadTime}}"}}" | Result: {{tmpCalc("{@Number|{leadTime}}")}}

Currency: "{{"{@Currency|~number}"}}" Example: "{{"{@Currency|234.56}"}}" | Result: {{tmpCalc("{@Currency|234.56}")}}
Example: "{{"{@Currency|{prices[supplier,kind].price}}"}}" | Result: {{tmpCalc("{@Currency|{prices[supplier,kind].price}}")}}

Math function

Not: "{{"{@Not|~variable}"}}" Example: "{{"{@Not|true}"}}" | Result: {{tmpCalc("{@Not|true}")}}
Example: "{{"{@Not|false}"}}" | Result: {{tmpCalc("{@Not|false}")}}
Example: "{{"{@Not|{consumable}}"}}" | Result: {{tmpCalc("{@Not|{consumable}}")}}

Sum: "{{"{@Sum|~number1|~number1}"}}" Example: "{{"{@Sum|1|2}"}}" | Result: {{tmpCalc("{@Sum|1|2}")}}
Example: "{{"{@Sum|{packageQuantity}|2}"}}" | Result: {{tmpCalc(" {packageQuantity}+2 = {@Sum|{packageQuantity}|2}")}}
Example: "{{"{@Sum|{packageQuantity}|{leadTime}}"}}" | Result: {{tmpCalc(" {packageQuantity}+{leadTime} = {@Sum|{packageQuantity}|{leadTime}}")}}

Math ( Sum '+',Difference '-',Product '*',Quotient '/',Remainder '%', Power '**'): "{{"{@Math|~sign|~number1|~number1}"}}" Example: "{{"{@Math|*|3|2}"}}" | Result: {{tmpCalc("{@Math|*|3|2}")}}
Example: "{{"{@Math|/|{packageQuantity}|2}"}}" | Result: {{tmpCalc("{packageQuantity}/2 = {@Math|/|{packageQuantity}|2}")}}
Example: "{{"{@Math|%|{leadTime}|{packageQuantity}}"}}" | Result: {{tmpCalc(" {leadTime} % {packageQuantity} = {@Math|%|{leadTime}|{packageQuantity}}")}}

Comparisons

IsNull (2 parameter): "{{"{@IsNull|~variable-nullable|~variable-alternative}"}}" Example: "{{"{@IsNull||alternative}"}}" | Result: {{tmpCalc("{@IsNull||alternative}")}}
Example: "{{"{@IsNull|exist|alternative}"}}" | Result: {{tmpCalc("{@IsNull|exist|alternative}")}}
Example: "{{"{@IsNull|{parent}|alternative}"}}" | Result: {{tmpCalc("{@IsNull|{parent}|alternative}")}}
Example: "{{"{@IsNull|{packageQuantity}|alternative}"}}" | Result: {{tmpCalc("{@IsNull|{packageQuantity}|alternative}")}} IsNull (3 parameter): "{{"{@IsNull|~variable-nullable|~if-not-null|~if-null}"}}" Example: "{{"{@IsNull||is null|is not null}"}}" | Result: {{tmpCalc("{@IsNull||is not null|is null}")}}
Example: "{{"{@IsNull|1|is null|is not null}"}}" | Result: {{tmpCalc("{@IsNull|1|is not null|is null}")}}
Example: "{{"{@IsNull|{parent}|is null|is not null}"}}" | Result: {{tmpCalc("{@IsNull|{parent}|is not null|is null}")}}
Example: "{{"{@IsNull|{packageQuantity}|is null|is not null}"}}" | Result: {{tmpCalc("{@IsNull|{packageQuantity}|is not null|is null}")}}

If (eval-string use js eval): "{{"{@If|~eval-string|~true-case|~false-case}"}}" Example: "{{"{@If|'true'=='false'|true case|false case}"}}" | Result: {{tmpCalc("{@If|'true'=='false'|true case|false case}")}}
Example: "{{"{@If|1==1|true case|false case}"}}" | Result: {{tmpCalc("{@If|1==1|true case|false case}")}}
Example: "{{"{@If|{packageQuantity}>9|{packageQuantity} is bigger then 9|{packageQuantity} isn't bigger then 9}"}}" | Result: {{tmpCalc("{@If|{packageQuantity}>9|{packageQuantity} is bigger then 9|{packageQuantity} isn't bigger then 9}")}}
Example: "{{"{@If|{reorderPoint}>9|{reorderPoint} is bigger then 9|{reorderPoint} isn't bigger then 9}"}}" | Result: {{tmpCalc("{@If|{reorderPoint}>9|{reorderPoint} is bigger then 9|{reorderPoint} isn't bigger then 9}")}}

Switch : "{{"{@Switch|~expression|~case-string-1:~value-case|~case-string-2:~value-case|...|~(default:)~value-default}"}}" Example: "{{"{@Switch|b|a:case a|b:case b|case default}"}}" | Result: {{tmpCalc("{@Switch|b|a:case a|b:case b|case default}")}}
Example: "{{"{@Switch|A|a:case a|b:case b|case default}"}}" | Result: {{tmpCalc("{@Switch|k|a:case a|b:case b|case default}")}}
Example: "{{"{@Switch|A|a:case a|A:case A|case default}"}}" | Result: {{tmpCalc("{@Switch|A|a:case a|A:case A|case default}")}}
SwitchInsensitive : "{{"{@SwitchInsensitive|~expression|~case-string-1:~value-case|~case-string-2:~value-case|...|~(default:)~value-default}"}}" Example: "{{"{@SwitchInsensitive|A|a:case a|B:case B|case default}"}}" | Result: {{tmpCalc("{@SwitchInsensitive|A|a:case a|B:case B|case default}")}}

Contains (3 parameter): "{{"{@Contains|~variable|~string-contained|~value-is-contained|~value-is-not-contained}"}}" Example: "{{"{@Contains|image.jpg|.jpg| contain .jpg }"}}" | Result: {{tmpCalc("{@Contains|image.jpg|.jpg| contain .jpg }")}}
Example: "{{"{@Contains|image/jpg|png|contain png }"}}" | Result: {{tmpCalc("{@Contains|image/jpg|png|contain png }")}}
Example: "{{"{@Contains|{productImage}|.png| {productImage} is png image }"}}" | Result: {{tmpCalc("{@Contains|{productImage}|.png| {productImage} is png image }")}}
Example: "{{"{@Contains|{productImage}|.jpg| {productImage} is jpg image }"}}" | Result: {{tmpCalc("{@Contains|{productImage}|.jpg| {productImage} is jpg image }")}}
Contains (4 parameter): "{{"{@Contains|~variable|~string-contained|~value-if-contain|~value-if-not-contain}"}}" Example: "{{"{@Contains|image.jpg|.jpg| contain .jpg |not contain .jpg }"}}" | Result: {{tmpCalc("{@Contains|image.jpg|.jpg| contain .jpg |not contain .jpg }")}}
Example: "{{"{@Contains|image/jpg|png|contain png|not contain png}"}}" | Result: {{tmpCalc("{@Contains|image/jpg|png|contain png|not contain png}")}}
Example: "{{"{@Contains|{productImage}|.png| {productImage} is png image | {productImage} isn't png image}"}}" | Result: {{tmpCalc("{@Contains|image/jpg|png|contain png|not contain png}")}}
